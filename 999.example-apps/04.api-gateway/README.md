# api-gatewayを作る

## 目標１：APIアクセスをルーティングする
プログラムの不足部分を完成させてルーティングができるようにする。
予め提供された設定ファイルを取り込み。
パスを設定通りにマッチさせてルーティングする。

## 目標2：ミドルウェアパターンを使って、レートリミットを実装する
- key-valueストアを起動し、アクセス回数を記録する。
- 設定した制限を超えたら10秒アクセス拒否する
- 手動テストで確認できればOK

## 目標3：動的にルーティングを変えられるようにする。
